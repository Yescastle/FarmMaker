[기본 시스템]
<조작>
- PlayerController : 플레이어를 조작하기 위한 스크립트. 이게 없으면 게임 플레이가 안된다.
  - 조작 : W A S D
  - UI 인터랙티브 : 마우스
  - 단축키 : 추후 지정

<플레이어 상태 관리>
- GameManager : 플레이어의 레벨, 체력, 노동력 등 현 상태를 관리한다.

<메뉴>
- UIManager : UI를 다루기 위한 스크립트. 집, 풍차, 밭, 인벤토리, 게임 옵션 등의 게임 내 인터랙티브를 다루는 스크립트.
  - 집 메뉴 : 잠자기, 집 강화 >>> +요리 >>> +보관함
  - 풍차 메뉴 : 재료 가공 >>> 작업용
  - 밭 메뉴 : 작물 심기
  - 인벤토리 : 농작물, 동물, 도구 임시 보관

<시간>
- TimeKeeper : Skybox 및 Directional Light를 관리하는 스크립트.

[콘텐츠 시스템]
<경작 및 포획>
- HarvestManager : 농장 내 경작을 다루기 위한 스크립트.					>> 엑셀 파일 참조.
- AnimalManager : 포획 전용 동물 (양, 소, 돼지, 닭 등)을 다루기 위한 스크립트.		>> 엑셀 파일 참조.
- EnemyManager : 밤에 등장하는 몬스터들을 관리하기 위한 스크립트.			>> 엑셀 파일 참조.

<건축>
- HouseArchitect : 집을 강화시키는 스크립트.
- WindmillArchitect : 풍차를 강화시키는 스크립트.
- FarmArchitect : 헛간, 외양간, 창고 등을 건설하는 스크립트.
- Farmer : 밭을 만들고 농작물을 심는 스크립트.

<인벤토리>
- InventoryManager : 인벤토리에 데이터를 저장하는 스크립트.
- ChestManager : 보관함(집 3단계)에 데이터를 저장하는 스크립트.


=========================================================
[기타 콘텐츠 시스템]
- EventManager : 모래바람, 새, 메뚜기 등 자연재해를 관리하는 스크립트.
- WeatherManager : 비, 눈, 번개 등 날씨를 관리하는 스크립트.
